Extraction Pipeline
Raw Email Input
Sender address, subject line, body text (plain or HTML), attachment flag, link count.
↓
HTML Stripping & Normalisation
HTML tags removed, whitespace normalised, URL tokens preserved, text lowercased for consistent vectorisation.
↓
URL Analysis
Regex extraction of all http/https URLs. Counts IP-based, shortener, HTTP-only and HTTPS links. Computes http_ratio.
↓
Domain Feature Engineering
Sender domain extracted via regex. Checked against known-legitimate list and suspicious TLD set. Domain length, digits, hyphens and age proxy computed.
↓
Semantic Feature Extraction
Scans body and subject for 24 urgency/deception phrases. Counts special characters and HTML tags as structural signals.
↓
Feature Vector Assembly
19 numeric features assembled into a DataFrame row. Passed through ColumnTransformer → HashingVectorizer (text) + StandardScaler (numeric) → ML classifier.
